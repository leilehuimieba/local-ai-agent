## 1. 使用规则

1. 本登记册覆盖 B-F 阶段关键风险。
2. 每条风险必须有触发条件、预警信号、缓解动作、失败回退动作。
3. 风险状态使用 `Open/InMitigation/Closed`。

## 3. 风险热区

1. B 阶段热区：R-01、R-02。
2. C 阶段热区：R-03、R-04、R-05、R-10。
3. D 阶段热区：R-06、R-07。
4. E 阶段热区：R-08、R-09。
5. F 阶段热区：R-11、R-12、R-15。

## 4. 风险触发阈值

1. 同类 P0 问题在 3 天内重复出现 2 次，必须触发阶段复盘。
2. 关键链路成功率连续两天低于 90%，必须暂停新功能推进。
3. 高风险动作漏拦截出现 1 次即触发紧急策略升级。
4. 新机安装失败率超过 10% 时不得进入发布候选。

## 5. 每周风险复盘清单

1. 上周新增风险条目。
2. 已关闭风险及证据。
3. 触发阈值命中情况。
4. 下一周风险控制动作。

## 6. Gate-A 风险判定

1. 关键风险已被识别并形成回退动作。
2. 风险热区与阶段目标已对齐。
3. `Gate-A/风险治理子项：通过（进入 B 前需补契约测试计划）`。
