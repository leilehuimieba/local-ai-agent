│          │                                                  │

### 1.3 关键改动

**删除的元素：**
- `TaskPageToolbar` — 运行/会话/工作区/状态标签/按钮组
- `ThreadHeader` — "任务主线程" + StatusPill
- `ThreadMeta` — "等待第一条任务" + 运行编号
- `BottomPanel` — 调查与执行轨迹（事件时间线）
- 所有状态记录卡片（Pending/Error/Confirmation 的 banner 样式）

**保留的元素：**
- 消息流（用户 + AI）
- Composer（改为流式）
- 确认弹窗（需要用户决策时）

**Composer 改造：**
- 从 `position: fixed` 改为普通流式定位
- 放在 `.messages` 容器内部，作为最后一条元素
- 空状态时居中显示
- 有消息时跟随在消息流下方

**消息气泡样式：**
- 用户：右对齐，圆角气泡，焦橙色边框，无头像
- AI：左对齐，圆角气泡，无头像，纯文本
- 有工具调用时：AI 消息下方展开工具调用过程（可折叠）

## 二、记录页

### 2.1 页面结构

```
┌─────────────────────────────────────────────────────────────┐
│  本地智能体                                    ≡  │
├──────────┬──────────────────────────────────────────────────┤
│          │  记录                                            │
│  ☰      │  ─────────────────────────────────────────────   │
│  ◉ 任务  │                                                  │
│  ≡ 记录  │  ● 2026-04-25 14:32                              │
│  📚 知识 │  智能体执行: hi                                    │
│  ⚙ 设置  │  [完成] 模型未下发工具调用，返回文本回答。          │
│          │  ─────────────────────────────────────────────   │
│  ＋     │                                                  │
│          │  ● 2026-04-25 14:30                              │
│          │  修改项目文件                                      │
│          │  [完成] 已修复 3 个文件，通过了测试。               │
│          │  ─────────────────────────────────────────────   │
│          │                                                  │
└──────────┴──────────────────────────────────────────────────┘
```

### 2.2 点击展开详情

```
│  ● 2026-04-25 14:32                              │
│  智能体执行: hi                                    │
│  [完成] 模型未下发工具调用，返回文本回答。          │
│  ─────────────────────────────────────────────   │
│  展开 ▼                                           │
│  ┌────────────────────────────────────────────┐  │
│  │ 工具调用: session_context                   │  │
│  │ 运行耗时: 2.3s                              │  │
│  │ 验证结果: 通过                               │  │
│  └────────────────────────────────────────────┘  │
```

### 2.3 关键改动

- 从 drawer 改回独立全页（`AppView = "logs"` 时全页渲染）
- 删除复杂的筛选面板（全部类型/全部审计/仅确认/风险/验证/治理）
- 删除 4 个统计卡片（稳定记录/异常收口/待确认/工具触达）
- 时间线改为简洁列表，每条记录一行摘要
- 点击展开详情（工具调用、验证、耗时等）
