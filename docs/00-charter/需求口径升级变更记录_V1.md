## 3. 本次口径升级的核心变化

本次新增并冻结为第一版正式能力域：

1. 启动本地程序。
2. 可控安装与验证能力：在命令行安装可行时，允许执行安装，并校验版本号或可执行路径。

原有第一版后置项保持不变：

1. 浏览器自动化主链路。
2. 桌面自动化大范围能力。
3. 多智能体。
4. 语音输入输出。
5. 自动 fallback / 自动测速。
