### 4.1 Prompt 收紧

已在 `prompt.rs` 中对 `render_project_answer_prompt()` 与 `render_context_answer_prompt()` 增加以下硬约束：

1. 只允许输出纯中文自然语言
2. 禁止输出 `tool_call`
3. 禁止输出 `minimax:tool_call`
4. 禁止输出 `workspace_read` 等工具调用字样
5. 禁止输出任何 XML、HTML、Markdown 标签
6. 禁止输出任何尖括号包裹内容
7. 禁止输出代码块、协议块、伪动作计划
8. 禁止出现“我先查看/读取文件”这类伪执行话术

### 4.2 结果收口

已在 `execution.rs` 中新增结果收口逻辑：

1. 对模型输出做敏感字样替换
2. 删除尖括号标签片段
3. 过滤伪动作行
4. 若清洗后结果不可用，则回落到安全中文兜底答复

### 4.3 本轮不扩范围

本轮没有改动：

1. provider 配置
2. 模型重试策略
3. 工具协议结构
