## 1. 文档目的

本文只回答 4 个问题：

1. 当前版本是否已经达到首版产品落地门槛
2. 《验收标准_V1》中的核心功能和核心场景是否已通过
3. 当前版本是否可以按“首版候选版本”对外或对内冻结
4. 还剩哪些非阻断遗留项

本文件不是新的开发任务书。
本文件是当前首版产品级总验收口径。

---

## 2. 总体验收范围

本次总体验收覆盖：

1. 启动与运行边界
2. 聊天与单任务主循环
3. 事件流与日志复盘
4. 命令执行与文件读写
5. 工作区与风险确认
6. 模式切换与模型切换
7. 记忆、知识检索、SQLite 主存储、思源外挂知识库
8. 前端产品形态与可观测性

本次明确不验收：

1. 浏览器自动化
2. 桌面自动化深链路
3. 多智能体
4. 语音输入输出
5. 自动 fallback
6. 自动测速
