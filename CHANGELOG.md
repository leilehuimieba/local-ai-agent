## [0.1.0] - 2026-04-27

### Added
- 本地智能体框架初始版本。
- 多模型 Provider 支持（SCNet、OpenAI、local-llama）。
- 本地记忆与知识沉淀（SQLite、思源笔记同步）。
- 任务执行与验证（Rust runtime-core + runtime-host）。
- Web 前端（React 19 + Vite）：聊天、任务、设置、知识库、日志、Release 向导。
- Go API 网关：HTTP 路由、设置管理、诊断、日志聚合。
- Hermes 重构总路线 A~H 全部完成并归档。
